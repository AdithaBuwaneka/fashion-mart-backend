• Review Designs
• Approve/Reject
• Manage Stock
• Handle Alerts

👥 Staff

• Process Orders
• Update Status
• Customer Support
• Handle Returns

👑 Admin

• Manage Users
• Assign Roles
• View Reports
• System Control

🔐 Key Approvals

Designer Role:
Customer → Request → Admin Approval
Design Approval:
Designer → Submit → Inventory Approval
Order Processing:
Customer → Order → Staff Processing
Returns:
Customer → Request → Staff Approval

Simple Flow: Designer Creates → Inventory Approves → Customer Buys → Staff Processes → Admin Manages
